    Class BaseMarketManagerService

    Abstract base class providing common market management functionality for the Kalshi trading bot. Handles market reset operations, interest score calculations, and provides protected methods for adding/removing markets based on various criteria. Serves as the foundation for both managed and unmanaged market management implementations. Now includes configurable queue limits and extracted target calculation service for improved testability.

    AddHighInterestMarkets(IKalshiBotContext, IKalshiAPIService, int, double)

    Adds the specified number of high-interest markets to the watch list. This method prioritizes existing market watches with high interest scores, then discovers new markets from active markets ordered by trading volume. Markets are added to the watch list only if they meet the minimum interest threshold. Handles both existing watches that need to be locked and new markets that need to be created.
