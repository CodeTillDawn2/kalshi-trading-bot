    Interface IKalshiAPIService

    Defines the contract for services that interact with the Kalshi trading platform API. This interface provides methods for fetching market data, managing positions and orders, retrieving account information, and accessing exchange status and schedules. Implementations should handle authentication, error handling, and data transformation.
