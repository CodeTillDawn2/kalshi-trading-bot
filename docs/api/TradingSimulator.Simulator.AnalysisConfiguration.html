    Class AnalysisConfiguration

    Configuration class for velocity discrepancy analysis parameters. This class contains all configurable parameters for the SimulatorReporting analysis engine, allowing fine-tuning of detection thresholds, performance settings, and output configuration. Parameters can be loaded from appsettings.json under the "SimulatorReporting" section.
