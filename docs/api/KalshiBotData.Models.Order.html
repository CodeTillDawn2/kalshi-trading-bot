    Class Order

    Represents a trading order in the Kalshi system with comprehensive execution tracking. This entity captures all aspects of an order's lifecycle including pricing, status, execution details, and performance metrics. Orders are the fundamental unit of trading activity and this entity provides complete visibility into order management and execution.
