    Class ExchangeScheduleExtensions

    Provides extension methods for converting between ExchangeSchedule model and ExchangeScheduleDTO, including nested maintenance windows and standard hours collections for comprehensive schedule data transfer. Includes performance metrics collection and batch transformation capabilities.
