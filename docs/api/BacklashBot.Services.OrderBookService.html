    Class OrderBookService

    Service responsible for managing order book data for Kalshi markets. Handles WebSocket order book events, processes snapshots and deltas asynchronously, maintains thread-safe access to order book data, and provides synchronization capabilities. Integrates with the broader trading bot ecosystem to ensure real-time order book updates.
