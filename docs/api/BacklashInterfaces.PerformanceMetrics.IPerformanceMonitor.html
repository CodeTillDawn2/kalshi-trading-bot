    Interface IPerformanceMonitor

    Interface for recording performance metrics across the KalshiBot system. Provides methods for each VisualType to ensure uniform handling of different metric types.

    Namespace: BacklashInterfaces.PerformanceMetrics
    Assembly: BacklashInterfaces.dll
    Syntax
    public interface IPerformanceMonitor
    Remarks

    All methods require the className to identify the source of the metric. Metrics are recorded through specific methods for each visual type to ensure proper categorization.
