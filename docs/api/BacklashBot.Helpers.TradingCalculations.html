    Class TradingCalculations

    Provides static utility methods for various trading calculations, including technical indicators, position metrics, and mathematical operations. This class is used throughout the trading bot for computing EMA, liquidation prices, fees, ROI, and other essential calculations. All methods are designed to be thread-safe and efficient for real-time trading operations.
