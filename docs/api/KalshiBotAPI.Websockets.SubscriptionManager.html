    Class SubscriptionManager

    Manages WebSocket channel subscriptions for real-time market data from Kalshi's trading platform. Handles subscription lifecycle, state tracking, confirmation processing, and queue management for reliable market data streaming with proper error handling and recovery mechanisms.
