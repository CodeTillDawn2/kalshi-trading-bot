    Class ResearchBus.EntryResearch

    Represents a single research entry containing detailed metrics about a trading entry point. This record captures comprehensive data about market conditions, entry timing, performance outcomes, and strategy parameters to enable thorough analysis of trading signal effectiveness and strategy optimization.
