    Class StandardHours

    Represents the standard operating hours configuration for the Kalshi exchange. This entity defines the regular trading schedule and time periods when the exchange is open for business. Standard hours are used to coordinate trading activities, schedule maintenance, and ensure proper market operation timing across the platform.
