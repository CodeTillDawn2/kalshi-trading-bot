    Class MarketWatch

    Represents the watch status and monitoring configuration for a specific market. This entity tracks which markets are being actively monitored by the trading bot, including interest scores, brain instance assignments, and activity metrics. MarketWatch entities are crucial for managing the bot's market coverage and optimizing resource allocation across different trading opportunities.
