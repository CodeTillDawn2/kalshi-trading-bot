    DumpCsv(string)

    Exports all logged research entries to a CSV file for external analysis. The CSV file includes a header row with column names and data rows ordered by score in descending order. For large datasets, parallel processing is used to build the CSV lines for improved performance. This facilitates analysis in spreadsheet applications or statistical tools for strategy optimization.
