    Class WeightSetMarket

    Represents the market-specific allocation and performance data within a weight set. This entity defines how individual markets are weighted within a portfolio strategy, tracking both the allocation percentage and the realized performance. It enables granular control over portfolio composition and performance attribution at the market level.
