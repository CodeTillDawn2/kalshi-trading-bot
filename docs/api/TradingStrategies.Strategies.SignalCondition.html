    Class SignalCondition

    Defines a condition based on a trading signal that triggers a specific trading action. This class combines a signal (technical indicator or market condition), a comparison operator, a threshold value, and the resulting action type to create rule-based trading logic. Signal conditions are used by strategies to evaluate market states and determine appropriate responses.
