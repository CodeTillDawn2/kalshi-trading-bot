    Class ConcealingBabySwallowPattern

    Represents a Concealing Baby Swallow candlestick pattern, a 4-candle bullish reversal pattern. Indicates a potential reversal from a downtrend to an uptrend. Requirements (Source: BabyPips, TradingView):

    • Occurs in a downtrend.
    • First two candles: Strong bearish (near Marubozu) with small wicks, small gap between.
    • Third candle: Bearish with upper wick overlapping second, slowing momentum.
    • Fourth candle: Bearish, engulfs third, closes below second low, signals reversal. Optimized for ML: Relative scaling based on lookback average range, gap relaxed to 0.
