    Class DojiPattern

    Represents a Doji candlestick pattern, a single-candle pattern indicating indecision. Suggests potential reversal or continuation depending on prior trend. Requirements (Source: Investopedia, DailyFX):

    • Single candle with a very small body (open and close nearly equal).
    • Shadows can vary, but total range should be significant relative to prior volatility.
    • Meaningful only after a clear trend, measured via TrendDirectionRatio. Optimized for ML: Relative scaling based on lookback average range, no volume dependency.
