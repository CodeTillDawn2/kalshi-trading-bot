    IsPattern(int, int, bool, CandleMids[], Dictionary<int, CandleMetrics>)

    Identifies a Tasuki Gap pattern, a three-candle continuation pattern. Bullish: Occurs in an uptrend; two bullish candles with a gap up between them, followed by a bearish candle that closes within the gap. Bearish: Occurs in a downtrend; two bearish candles with a gap down between them, followed by a bullish candle that closes within the gap. Indicates continuation of the prior trend after a brief pullback. Source: https://www.investopedia.com/terms/t/tasukigap.asp
