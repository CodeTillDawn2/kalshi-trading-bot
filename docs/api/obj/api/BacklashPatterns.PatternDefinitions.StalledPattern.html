    Class StalledPattern

    Identifies a Stalled Pattern, a three-candle reversal pattern signaling a weakening uptrend, often indicating a potential top or pause in bullish momentum.

    Requirements (Source: BabyPips, "Stalled Pattern"):

    • First candle: Strong bullish candle with a large body.
    • Second candle: Bullish candle with a smaller body, closing above the first.
    • Third candle: Small body (bullish or bearish), closing below the second but near the first, showing momentum stalling.
    • Occurs in an established uptrend.
    • Suggests a potential reversal or consolidation.
