    BaseName

    Represents a Three Stars in the South pattern, a rare bullish reversal pattern in a downtrend.

    • Three bearish candles with decreasing size and specific wick/low/high relationships, indicating slowing bearish momentum before a potential reversal. Requirements sourced from: https://www.tradingview.com/education/threestarsinthesouth/
