    Class DatabaseLoggingQueue

    Background service that asynchronously processes log entries from a concurrent queue and persists them to the database with batching support for improved performance. Also manages error handler queues for warnings and errors. Supports both regular application logging and overseer-specific logging contexts. Provides metrics and monitoring for queue depth and processing performance.
