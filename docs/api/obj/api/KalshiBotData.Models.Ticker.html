    Class Ticker

    Represents real-time ticker data for a Kalshi market, capturing price and volume information at a specific point in time. This entity stores the most recent market data including bid/ask prices, trading volume, and open interest. Ticker data is crucial for real-time trading decisions and market analysis, providing the current state of market pricing and liquidity for individual contracts.
