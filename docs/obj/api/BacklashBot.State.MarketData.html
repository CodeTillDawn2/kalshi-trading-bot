    MarketData(MarketDTO, ILogger<IMarketData>, ITradingCalculator, IOrderbookChangeTracker, IOptions<CalculationConfig>)

    Initializes a new instance of the MarketData class for a specific market. Sets up all data collections, dependencies, and initial state for market data tracking. Initializes candlestick dictionaries for minute, hour, and day timeframes, and configures calculation parameters from the provided configuration.
