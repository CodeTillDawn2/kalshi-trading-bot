    Class ReportGenerator.SimulationEventLog

    Represents a single event log entry from a trading simulation, capturing the complete state of the trading environment at a specific point in time. This data structure is used to record and analyze the progression of trading activities, market conditions, and strategy decisions throughout a simulation run.
