    IsPattern(Dictionary<int, CandleMetrics>, int, int, bool, CandleMids[])

    Identifies a Separating Lines pattern, a two-candle continuation pattern. Requirements (source: TradingView, Investopedia):

    • Bullish: Occurs in a downtrend; first candle is bearish, second is bullish, both open at nearly the same price.
    • Bearish: Occurs in an uptrend; first candle is bullish, second is bearish, both open at nearly the same price. Indicates: Continuation of the existing trend (bullish or bearish) with a strong move after a brief pause.
