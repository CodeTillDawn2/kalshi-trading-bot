    Class HangingManPattern

    The Hanging Man is a single-candle bearish reversal pattern appearing after an uptrend.

    • Small body, long lower wick, minimal upper wick.
    • Suggests selling pressure after a rise, potential top. Indicates: Bearish reversal if confirmed by subsequent decline. Source: https://www.investopedia.com/terms/h/hangingman.asp
