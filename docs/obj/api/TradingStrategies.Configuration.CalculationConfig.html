    Class CalculationConfig

    Configuration class containing all technical indicator parameters used throughout the trading bot. These settings control the calculation periods and thresholds for various technical analysis indicators that drive trading decisions and market analysis. Values are injected via dependency injection and used by MarketData and TradingCalculator classes for real-time indicator computation.
