    Namespace BacklashBot.Helpers

    Classes

    SnapshotDiscrepancyValidator

    Provides static methods for validating market snapshots to detect data discrepancies and ensure data integrity. This validator is used to filter out invalid snapshots before processing or saving them, preventing downstream issues from corrupted or inconsistent market data.

    SnapshotDiscrepancyValidator.ValidationResult

    Represents the result of a snapshot validation operation, containing flags for specific discrepancy types and an overall validity indicator.

    TradingCalculations

    Provides static utility methods for various trading calculations, including technical indicators, position metrics, and mathematical operations. This class is used throughout the trading bot for computing EMA, liquidation prices, fees, ROI, and other essential calculations. All methods are designed to be thread-safe and efficient for real-time trading operations.
