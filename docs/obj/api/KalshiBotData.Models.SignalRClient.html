    Class SignalRClient

    Represents a client connected to the system via SignalR for real-time communication. This entity tracks client connections, authentication, and connection state for managing real-time data distribution to dashboards, overseer systems, and other connected clients. It enables secure, authenticated real-time communication between the trading bot system and its clients.
