    Class InNeckPattern

    The In Neck Pattern is a two-candle continuation pattern in a downtrend.

    • First candle bearish, second bullish with close near first close.
    • Indicates temporary pause in downtrend, likely to continue downward. Source: https://www.babypips.com/learn/forex/in-neck-pattern
