    Class KalshiBotContext

    Entity Framework DbContext implementation for the Kalshi trading bot system. Provides comprehensive data access layer for managing trading entities including markets, events, series, snapshots, brain instances, and various trading-related data. Implements IKalshiBotContext interface for dependency injection and testing.

    public class KalshiBotContext : DbContext, IInfrastructure<IServiceProvider>, IDbContextDependencies, IDbSetCache, IDbContextPoolable, IResettableService, IAsyncDisposable, IKalshiBotContext, IDisposable
    Remarks

    This context manages the complete lifecycle of trading data, including:

    • Market and event data synchronization
    • Trading snapshots and historical data
    • Brain instance management and coordination
    • Order and position tracking
    • SignalR client management
    • Logging and audit trails
    • Weight set and strategy management
